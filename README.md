# PhonePe Transaction Insights

This is a Data Engineering and Analytics project based on PhonePe transaction data.

## Project Objective

The objective of this project is to extract, clean, transform, store, analyze, and visualize PhonePe transaction data to understand transaction trends, user engagement, geographical performance, and insurance-related insights.

## Skills Used

- Python
- Pandas
- SQL
- SQLite/MySQL
- Data Cleaning
- ETL Pipeline
- Data Visualization
- Streamlit
- Business Analysis

## Project Workflow

1. Data extraction from PhonePe dataset
2. Data cleaning and transformation
3. SQL database and table creation
4. Data loading into SQL tables
5. SQL-based analysis
6. Python visualizations
7. Streamlit dashboard creation
8. Business insights and recommendations

## Key Business Use Cases

- Customer segmentation
- Fraud pattern analysis
- State-wise and district-wise transaction analysis
- Payment category performance
- User engagement analysis
- Insurance transaction insights
- Marketing optimization
- Trend analysis
